Parse checks (@supports)

  • supports view-transition-name: foo
  • ident() is invalid
  • ident(myident) is valid
  • ident(3 "myident") is valid
  • ident(rgb(...)) is invalid
  • ident(5px) is invalid

Computed value checks (JS)

  • ident("myident" 3) -> myident3
  • ident(-- myident) -> --myident
  • ident("--myident" random(element-shared, 0, 5)) on view-transition-name
  • ident("--myident" random(element-shared, 0, 5)) on container-name
